# Capstone_Breast_Cancer_Prediction

## required installs
  - install python version - 3.0.0 or later
  - install pip
    - pip install pandas
    - pip install numpy
    - pip install matplotlib
    - pip install seaborn
    - pip install scikit-learn
    - pip install flask
